Write and Explain important attributes for the following tags 

1.body  2.form  3.textarea  4.iframe  5.applet

1.body: This tag is used for defining the body of our web page.It contains all the elements and contents of our webpage.

2.form: This tag is used collecting the input from the user through a form and pass data to required directory.

3.textarea: This tag is used for collecting user text input .It can be used to collect large amount of data and it's size can be modified using cols and rows attributes.

4.iiframe: This tag is used within a document where we can view another document that was not a part of the same page.This tag is generally used to embed one web-page in another web-page.

5.applet: This tag is used to embedd the java applets in html page.
