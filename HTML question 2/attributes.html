The attributes for body tag:

    1)background: used to set image as background
    2)bgcolor   : used to set color as background
    3)alink     :used to set the color of selected link.
    4)vlink     :used to set the color of visited link.
    5)link      :used to set the color of unvisited link.
    4)text      :used to adjust the color of the text.

The attributes for form tag:

  1)method: specifies which method is used to submmit the data of the form
  2)target: It specifies where the response should be displayed.
  3)action: It specifies the further process that should be done or the path of next operations after submitting the form.
  4)name  : It specifies the name of the form.
