The type of bullets in lists can be modified by using type attribute

Example for ordered lists bullets

    Animals:
  1. Cat
  2. Rat
  3. Dog
    Animals:
  1. Cat
  2. Rat
  3. Dog
    Animals:
  1. Cat
  2. Rat
  3. Dog
    Animals:
  1. Cat
  2. Rat
  3. Dog
    Animals:
  1. Cat
  2. Rat
  3. Dog

Example for types of bullets in un-ordered bullets

    Animals:
  • Cat
  • Rat
  • Dog
    Animals:
  • Cat
  • Rat
  • Dog
    Animals:
  • Cat
  • Rat
  • Dog
